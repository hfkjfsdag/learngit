　　在上次与孙斌老师的谈话之后，我回去阅读了《机器学习实践——测试驱动的开发方法》这本书，它主要讲解了七个主要的机器学习的算法，同时对于每个算法都提供了相应的代码示例，有助于我们进一步的理解。
　　第一个算法是K近邻分类，这个算法给了我很大的启发，这是一个十分实用的算法，可以用于生活中的许多方面。例如书中提到的例子，判断某人的购物爱好，基于邻居的居住幸福度以及人脸识别。这个看似很简单的算法让我第一次真正认识到什么是机器学习，并对它有了一些模糊的印象。这个算法主要是通过找到一个合适的K值，来实现对所有目标的选择，例如在历史购物单中，通过调取近5天来该顾客的购物记录来判断他最近的消费爱好，又例如通过最近的邻居是否幸福判断该处的幸福度。所以，最关键的是找到合适的K值，这里有三种方法，猜测，使用启发式策略，通过算法优化。猜测K值需要我们对目标本身具有较为深入的了解，再根据自己的猜测选择一个合适的K值，但是这个方法并不实用。然后就是启发式策略，我比较推崇这种方法，它主要包括三条策略。1.当分类问题中只涉及两个类别时，不要将K取为偶数。2.K的值应不小于类别总数加1。3.为避免出现噪声，K的值应该足够小。
通过这三个策略，我认为可以较好地确定出来一个K值，毕竟他提供了一个大致的范围，对于初学者来说还是比较合适的。最后还有K的选择算法，有关这方面，书中并没有详细讲述，所以我也不好评价。在本章的最后，该书提供了一个人脸识别的实例。但是该例是用Ruby语言编写的，所以我只是看懂了一个大概，但是还是深刻感受到这个算法的厉害之处。没想到人脸识别可以通过这样的算法实现。这个例子开阔了我的眼界，以前一直认为人工智能的算法实现都是十分高大上的，没想到有的原理竟是如此简单。
